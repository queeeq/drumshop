# Drumshop

## This is a drum shop app that allows users to view and purchase bass drum beaters. The admin can add products to the shop, edit, and delete products from the shop.

### How to run:

- Clone the repo
- Run `npm install`
- Run `npm run dev`
- Open `localhost:3000` in your browser
- Enjoy!

### Tech stack:

- Vite
- React
- SASS
- MUI
- React Router
- Supabase
